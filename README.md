stormwater-culvert-screening-dataset
Self-created synthetic input files for a stormwater culvert asset-screening and maintenance-priority reconciliation task.

Description
This repository contains self-created synthetic input files for controlled evaluation of rule-based engineering reconciliation. The files are designed to support cross-file reconciliation, source hierarchy application, calculation auditability, validation-rule checking, maintenance-priority classification, and traceable reporting.
The files do not contain real municipal records, real field measurements, real infrastructure asset records, private property information, public-agency datasets, or confidential operational data. No third-party tables, copyrighted manual text, public-agency asset records, or municipal datasets were copied into these files.
